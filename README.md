# OBGYN Master

An interactive, evidence-based educational platform for mastering obstetrics and gynecology topics.

## Features

- **Comprehensive Coverage**: Essential OB/GYN topics including ectopic pregnancy, preeclampsia, GDM, placental disorders, PPH, labor progression, and more
- **Interactive Tools**: Clinical calculators and interactive learning modules
- **Case-Based Learning**: Realistic clinical scenarios to apply your knowledge
- **Self-Assessment**: MCQ questions with detailed rationales
- **Mobile-First**: Responsive design optimized for learning on any device
- **Offline Capable**: Progressive Web App with service worker support
- **Dark Mode**: Eye-friendly theme options

## Topics Covered

1. Ectopic Pregnancy - Î²-hCG interpretation and triage
2. Preeclampsia & Eclampsia - Severity assessment and management
3. Gestational Diabetes - OGTT interpretation
4. Placental Disorders - Abruption vs Previa
5. Intrahepatic Cholestasis of Pregnancy (ICP)
6. Postpartum Hemorrhage - 4 Ts framework
7. Early Pregnancy Loss - Classification and management
8. Labor Progression - Normal vs abnormal patterns
9. Gynecologic Emergencies - Ovarian torsion and PID

## Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Zustand (state management)
- Recharts (data visualization)
- PWA support with workbox

## Educational Disclaimer

This application is for educational purposes only and uses synthetic data. It is NOT intended for clinical use. All clinical decisions must be made by qualified healthcare providers following current evidence-based guidelines and institutional protocols.
